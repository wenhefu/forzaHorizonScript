# 地平线6 刷分助手

用虚拟手柄在 EventLab 自定义图里自动刷技能点：自动循环「踩住油门前进 → X 重开 → A 确认 → A 开始」。
**不改任何游戏文件**，只模拟一个 Xbox 手柄。

## 一次性准备
1. 安装 **ViGEmBus 驱动**（虚拟手柄的系统驱动）。没装的话点开始会有提示。
2. 只有想改代码或自己打包的人，才需要再装 **Python 3**。

## 怎么用（拿到 exe 的朋友）
1. 进游戏，选好刷分用的 EventLab 图，把斯巴鲁 22B 摆在起点。
2. 打开 `Forza6Helper.exe`。
3. 按需在界面里改这几个值：
   - **启动倒计时（秒）**：留几秒让你切回游戏
   - **每圈前进时间（秒）**：你一圈大概多久，默认 44
   - **总运行时间（分钟）**：0 = 一直刷到你点停止
4. 推荐让游戏保持前台，按 **Ctrl+Alt+F8** 开始/停止；也可以点 **开始**，在倒计时内 **切回地平线**。
5. 不刷了点 **停止**，或直接关窗口。

日志在 `logs\forza6helper.log`，界面里的 **打开日志** 可以直接查看。

## 想边刷边用电脑（失焦不暂停）
- **先试这个**：游戏画面设置里改成 **无边框窗口**，多数情况下切走也不会暂停，且零风险。
- 还不行：勾界面里的 **「失焦时尝试保持运行」**（实验性，可能有用可能没用）。
- 说明：本软件**不做**那种注入游戏进程的"强力"失焦工具——地平线带反作弊，注入有封号风险。

## 自己打包成 exe（开发者）
- 在 **Windows** 上双击 `build.bat`，生成 `dist\Forza6Helper.exe`，发给朋友即可。
- 注意：exe 必须在 Windows 上打包；朋友的电脑也要各自装一次 ViGEmBus。

## 提醒
- 自动刷分违反 Forza 服务条款，理论上有封号风险，**自行斟酌**。
- 仅用于自己的单机进度。
