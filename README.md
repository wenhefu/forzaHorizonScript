# 地平线6 刷分助手

用虚拟手柄在 EventLab 自定义图里自动刷技能点：自动循环「踩住油门前进 → X 重开 → A 确认 → A 开始」。
**不改任何游戏文件**，只模拟一个 Xbox 手柄。

## 一次性准备
1. 安装 **ViGEmBus 驱动**（虚拟手柄的系统驱动）。没装的话点开始会有提示。
2. 只有想改代码或自己打包的人，才需要再装 **Python 3**。

## 怎么用（拿到 exe 的朋友）
1. 进游戏，选好刷分用的 EventLab 图，把斯巴鲁 22B 摆在起点。
2. 打开 `Forza6Helper.exe`。
3. 默认使用 **模式一 · 刷技能点（EventLab）**。按需在界面里改这几个值：
   - **启动倒计时（秒）**：留几秒让你切回游戏
   - **每圈前进时间（秒）**：只给旧的前台计时模式用；智能识别模式不靠它判断完赛
   - **总运行时间（分钟）**：0 = 一直刷到你点停止；刷技能点模式默认就是 0
4. 推荐让游戏保持前台，按 **Ctrl+Alt+F8** 开始/停止；也可以点 **开始**，在倒计时内 **切回地平线**。
5. 不刷了点 **停止**，或直接关窗口。

日志在 `logs\forza6helper.log`，界面里的 **打开日志** 可以直接查看。

## 五种运行模式（界面顶部选择）
- **模式一 · 刷技能点（EventLab）**：不盲计时。程序在内存里截取 Forza 窗口并识别画面：开始菜单先确认光标在“开始竞赛赛事”再按 A，比赛中保持油门，结果页按 X，重开确认按 A；如果识别到暂停菜单会先按 B 返回。截图不保存到硬盘。
- **模式二 · 买车加点（先买22B）**：会打开暂停菜单，进入车辆页，进入车展，前往制造商，选择斯巴鲁 22B，走默认设计/颜色并确认购买；买完后返回车辆页，进入升级与调校里的车辆熟练度，按固定路径买到抽奖精灵，再回到购买与出售页继续循环。买车流程会用 OCR 作为二次确认来识别关键文字，例如“购买新车与二手车”“车展”“制造商”“斯巴鲁”“控制器未连接”等。
- **模式三 · 买车+刷分组合（实验）**：先跑模式二买车加点；检测到“技术点数不足/不够购买额外加成”后，自动按 A/B 退回自由漫游，打开暂停菜单，进入创意中心 -> EventLab -> 赛事 -> 我的收藏；进入 EventLab 开始赛事菜单后交给模式一继续刷技能点。
- **模式四 · 前台计时（兜底）**：旧逻辑。游戏保持前台，按设定秒数跑，失焦会暂停脚本计时并尝试切回。
- **模式五 · 后台尝试（实验，不保证）**：想边刷边用电脑时选它。会尝试"保持活动" + 建议你把游戏设成 **无边框窗口**。但地平线很可能失焦就暂停，**本模式不保证有效**。
- 为什么不做"强力后台"：nofocusloss 那类工具靠**注入游戏进程**，而地平线常驻在线 + 首发反作弊，**篡改进程会被永久封号**（工具作者自己也写明 multiplayer 不能用）。所以本软件坚决不注入。

## 当前验证状态
- 模式一和模式二都已经经过实机流程验证，可以作为稳定基线继续开发。
- 模式二在技能点不足时会停止，不会继续误买或误加点。
- 模式三已接入组合编排，当前会从点数不足弹窗切到 EventLab 刷技能点；完整“刷够点后再回买车”的自动回环还需要后续确定补点结束条件。
- 详细 checkpoint 见 `docs\2026-05-25-stability-checkpoint.md`。

智能识别默认截图频率：
- 菜单/确认页：约 0.75 秒一次
- 刚进入比赛的前 10 秒：约 2 秒一次
- 比赛稳定行驶后：约 5 秒一次

## 自己打包成 exe（开发者）
- 在 **Windows** 上双击 `build.bat`，生成 `dist\Forza6Helper.exe`，发给朋友即可。
- 注意：exe 必须在 Windows 上打包；朋友的电脑也要各自装一次 ViGEmBus。

## 提醒
- 自动刷分违反 Forza 服务条款，理论上有封号风险，**自行斟酌**。
- 地平线是常驻在线的共享世界（不是纯单机），在线遥测较严，请知悉风险。
